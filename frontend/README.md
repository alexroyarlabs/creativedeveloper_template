# Characters Counter SPA

Modern React + Tailwind single-page app that provides a live character counter with channel presets, insights, and supporting pages.

## Stack
- React 18 with functional components and React Router
- Vite bundler
- Tailwind CSS for styling

## Getting started
```bash
cd frontend
npm install
npm run dev
```
Then open the printed local URL (default `http://localhost:5173`).

## Build for production
```bash
npm run build
npm run preview
```

## Key scripts
- `npm run dev` – start the dev server
- `npm run build` – production build
- `npm run preview` – preview built app
