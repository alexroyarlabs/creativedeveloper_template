# Bingo familiar – SPA

Aplicación React con Vite y Tailwind para organizar partidas de bingo familiar sin papeles.

## Scripts

- `npm install`
- `npm run dev` abre el entorno de desarrollo en `http://localhost:5173`.
- `npm run build` genera la versión de producción.
- `npm run preview` sirve el build generado.

## Estructura

- `src/pages` contiene las vistas: inicio, crear partida, cartón demo y cómo funciona.
- `src/components` agrupa layout, navegación y UI reutilizable (botones, badges, cards).
- `tailwind.config.js` define la paleta crema/menta/coral y sombras suaves.

No necesita cuenta, backend ni configuración externa: es demo estática con datos de ejemplo.
